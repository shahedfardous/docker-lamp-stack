# Simple Apache LAMP Stack
>A Simple LAMP Stack to start projects faster.
<br/>
<br/>

## Instruction
1. Clone the git and modify the `docker-composer.yml` file with your own details.
2. Run `docker-compose up -d` and you are good to go.
